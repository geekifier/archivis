## crates/archivis-kepub/src/assets/kobo.js

Reference: https://github.com/pgaskin/kepubify (MIT)

A minimal Kobo reader stub used by KEPUB on-the-fly conversion.
Vendored as a placeholder so KEPUB-aware reader features can attach when
the file is loaded on a Kobo device. The runtime behavior is supplied by
the Kobo device's reader; no executable Kobo logic is reproduced here.
