# Introduction

Archivis is a self-hosted ebook collection manager built for people who want full control over their library. It combines import automation, metadata extraction, and a modern web interface into a single binary that you can run anywhere.

## Key Features

- **Import flexibility** — Upload ebooks or scan directories for bulk import
- **Format support** — Automatic format detection and metadata extraction for EPUB, MOBI, and PDF
- **Background processing** — Imports, metadata resolution, and conversions run as async tasks with real-time progress tracking
- **Library management** — Grid and list views with full-text search, sorting, and filtering
- **Metadata editing** — Edit book metadata with autocomplete support
- **REST API** — Full OpenAPI documentation and programmatic access
- **Single binary** — Embedded SQLite database, no external services required
- **Theme support** — Dark and light mode
